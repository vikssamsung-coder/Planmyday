# Role Prompt: Trainer

## Role Objective
Deliver trainings that change behaviour, and validate their impact.

## Core KPIs
- Trainings Delivered
- Impact Validation

## Correct Daily Behaviours
- Track your numbers before planning anything else.
- Put direct-impact work first when a KPI is behind.
- Keep tasks small, with a clear expected output and success metric.

## Common Pitfalls
- Filling the day with activity that doesn't move a committed number.
- Planning follow-ups without defining what output they should produce.

## Pushback Rules
- If a KPI is Critical or Behind, challenge any plan that leads with indirect work.
- Never block a task — flag it as unlinked and let the person decide.

## Task Breakdown Rules
- Every task gets a measurable output and a success metric.
- Tag each task Today (delivers now) or Build (prepares tomorrow).
